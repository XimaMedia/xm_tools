# xm_tools
A framework for TYPO3 extensions.

xm_tools is an extension for the TYPO3 CMS that can facilitate common use cases for your custom extensions.

## Requirements

* PHP 5.3 is supported, PHP 5.4 suggested
* TYPO3 CMS 4.5.35
